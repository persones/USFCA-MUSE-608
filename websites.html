Accessabilty

  • Page title: <title> element in <head>
  • Image text alternatives: <img src="image.png" alt="here's where the alternative text goes">
  • Headings: use <h1>...<h6>
  • Use high-contrast color pallette
  • Allow text resize
  • Allow keyboard navigation (tab)
  • Use proper form tags:
